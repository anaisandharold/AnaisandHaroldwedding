# Anaïs & Harold — A Winter Premiere

## Preview
Open `index.html` in a browser.

## Before publishing
In `script.js`, replace `YOUR_EMAIL@example.com` with the email address that should receive RSVPs.

## Uploading an update to GitHub
1. Open your wedding repository on GitHub.
2. Click **Add file** → **Upload files**.
3. Drag `index.html`, `styles.css`, and `script.js` onto the page.
4. When GitHub warns that files already exist, continue: the new copies will replace them.
5. Click **Commit changes**.
6. Wait a few minutes, then refresh your GitHub Pages website.

The site uses no paid services and no build tools.
